Local installation

If you would like to try the app locally clone the repository 

git clone git@github.com:lmmentel/mendeleevapp.git

Dependencies

  • flask
  • gunicorn
  • bokeh
  • numpy
  • pandas
  • mendeleev

If you want to run it locally then I recommend installing foreman too.

Data

The data used here is available through the mendeleev python package that contains SQLite database with properties of elements, ions and isotopes. If you want to know more about the data sources take a look at the documentation of mendeleev data.
